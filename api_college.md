## <a name="college_list-01"/>讲师列表
地址：&act=lecturer_list
### 接口输入
提交方式：GET
### 描述
讲师列表

参数名     |必填  |类型   |参数                        |说明
----------|-----|-------|---------------------------|----------------
page      |  N  |string |                           |获取第N页(默认为1)

### 接口输出

字段名      | 必有 |   类型   |  示例  |  说明
-----------|------|---------|-------|---------
code       |  是  |  int    | 见示例 | 返回数据
msg        |  是  |  string | 见示例 | 返回数据
data       |  是  |  list   | 见详参 | 返回数据

#### 返回data结构
字段名      |必有  |类型   |示例                  |说明
-----------|------|------|---------------------|------------
lecturer_id|  是  |string |                    |讲师ID
lecturer_name| 是 |string |                    |讲师名字
file_path  |  是  |string |                    |图片路径
