## <a name="find_list-01"/>发现-动态
地址：&act=index
### 接口输入
提交方式：GET
### 描述
发现模块的动态

参数名     |必填  |类型   |参数                        |说明
----------|-----|-------|---------------------------|----------------
page      |  N  |string |                           |获取第N页(默认为1)

### 接口输出

字段名      | 必有 |   类型   |  示例  |  说明
-----------|------|---------|-------|---------
code       |  是  |  int    | 见示例 | 返回数据
msg        |  是  |  string | 见示例 | 返回数据
data       |  是  |  list   | 见详参 | 返回数据

#### 返回data结构
字段名      |必有  |类型   |示例                  |说明
-----------|------|------|---------------------|------------
title      |  是  |string |                    |标题
content    |  是  |string |                    |内容
article_id |  是  |string |                    |动态ID
file_path  |  是  |string |                    |封面图片路径
